# Archive

Точечные брифы задач, зафиксированные на определённый момент времени, содержимое которых полностью реализовано либо устарело.
Хранятся для исторической справки (почему было принято то или иное решение, как выглядел
исходный план) -- не поддерживаются, могут ссылаться на пути, которых больше не существует.

**Актуальный статус всегда находится в [`../../BACKLOG.md`](../../BACKLOG.md).** Если
вам нужно передать задачу сессии Gemini/агента, напишите новый самодостаточный
бриф, а не оживляйте один из этих.

- `GEMINI-TASKS.md` -- бриф механической работы (конфигурации скраперов venue/artist).
  Архивирован 2026-07-21: все задачи выполнены.
- `SMART-AGENT-TASKS.md` -- пункты, требующие экспертной оценки, по безопасности/архитектуре
  (S1-S5). Архивирован 2026-07-21: S1-S4 реализованы, S5 заменён миграцией шардирования
  `data/artists/`.
