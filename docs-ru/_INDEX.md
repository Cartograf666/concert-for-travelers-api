# Документация на русском

Разовый перевод-снимок проектной документации на русский, сделан 2026-07-21.
**Это не поддерживаемая копия** — при следующих обновлениях английских
файлов эта папка не синхронизируется автоматически и начнёт расходиться с
оригиналом. Источник истины всегда — английские файлы в корне репозитория
и в `docs/`.

Не переведены (намеренно): `LICENSE` (юридический текст, английский —
стандартная форма для OSS-лицензий), `docs/openapi.yaml` (машиночитаемый
API-контракт для внешнего consumer-приложения и тулинга), `.ai/*.yaml`
(манифест архитектуры, схема провалидирована как есть, ключи должны
оставаться на английском).
